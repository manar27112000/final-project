
# 🌐 Final Project – Doccure Healthcare Template  

🚀 A responsive front-end healthcare and pharmacy website developed as a final project.

This project represents a structured template for a healthcare system that includes home page, pharmacy listing, pharmacy details, and contact form.

---

## ✨ Project Overview  

This project includes the following pages:

✅ Home Page – Introduction to the website and services  
✅ Contact Us – Form for users to send messages  
✅ Pharmacy Page – Displays a list of pharmacies  
✅ Pharmacy Details Page – Shows detailed information about each pharmacy  

Built using clean structure and reusable components.

---

